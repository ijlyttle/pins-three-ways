2.2 Remote board

I donâ€™t think I will create a Python package; maybe I can just put together a quick script to convert a pins manafest into a dictionary.

import requests
import yaml

# read file, parse into manifest
url_root = "https://ijlyttle.github.io/pins-three-ways/pins"
req = requests.get(f"{url_root}/pins.txt")
manifest = yaml.safe_load(req.text)
# use the most-recent version
pin_paths = {}
for key, value in manifest.items():
    pin_paths[key] = max(value)
